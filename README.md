# Skazo

Проект сайта с рассказами и сказками для детей: читалка “как книга” + менеджерский контур для полуавтоматической подготовки контента (текст → сцены → иллюстрации → ревью → публикация).

Документация: см. `docs/README.md`.

## Dev: качество кода

Установка dev-зависимостей и хуков:

- `uv sync --group dev`
- `uv run --group dev pre-commit install`

Проверка:

- `make lint`
