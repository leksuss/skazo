# Глоссарий

- **Story (Произведение)**: рассказ/сказка как единица каталога.
- **Source (Источник)**: файл/URL, откуда получен исходный текст.
- **RawText**: текст “как есть” после извлечения.
- **NormalizedText**: текст после нормализации (правилами) и, опционально, корректуры.
- **Block (Блок)**: минимальная единица структурированного текста (обычно абзац).
- **Scene (Сцена)**: смысловой фрагмент произведения, состоящий из нескольких блоков.
- **Anchor (Якорь)**: привязка ассета к месту в тексте (после блока/к сцене).
- **Asset (Ассет)**: медиа-результат (изображение/видео), связанный с якорем/сценой.
- **Variant (Вариант)**: одна из попыток/версий ассета.
- **GenerationTask**: фоновая задача генерации ассета во внешнем AI API.
- **Preset (Пресет)**: набор правил (сколько сцен/иллюстраций, лимиты, форматы).
- **Publish**: перевод версии произведения в состояние доступности читателям.
- **Idempotency (Идемпотентность)**: повторный запуск задачи не создаёт дубликаты и приводит к тому же результату.
